## การทำงานของฟังก์ชัน

### 1. อ่านข้อมูลจากโครงสร้างต้นไม้

```javascript
const treeData = this.treeNodes;
```

โครงสร้างต้นไม้มีลำดับชั้น:
```
อาชีพ (Career)
└── คุณสมบัติ (Qualification)
    ├── ความสำคัญ (Importance)
    ├── ค่าเป้าหมาย (Target)
    ├── การพัฒนาทักษะ (Development)
    │   └── แผน (Plan) [1..n]
    │       ├── ชนิด
    │       ├── เรื่อง
    │       ├── ช่องทาง
    │       ├── ความสำคัญ
    │       ├── วันเริ่ม
    │       └── วันสิ้นสุด
    └── การประเมินตนเอง (Assessment Group)
        └── การประเมิน (Assessment) [1..n]
            ├── วันที่ประเมิน
            ├── ผลการประเมิน
            └── แผน (Assessment Plan)
                └── แหล่งอ้างอิง (Reference Group)
                    └── รายละเอียด (Reference Detail) [1..n]
```

### 3. จัดรูปแบบ Excel

**หัวตาราง:**

- ตัวอักษร: ตัวหนา, สีขาว, ขนาด 11pt
- จัดกึ่งกลาง
- ความสูง: 40px
- Wrap text: เปิด

**ข้อมูล:**
- เส้นขอบ: บางทุกเซลล์
- จัดชิดบน, ชิดซ้าย
- Wrap text: เปิด
- ความสูงแถว: ปรับอัตโนมัติ

## กรณีพิเศษ

### 1. ไม่มีแผนและการประเมิน
```
[อาชีพ, คุณสมบัติ, ความสำคัญ, เป้าหมาย, -, -, -, -, -, -, -, -, -]
```

### 2. มีแผนแต่ไม่มีการประเมิน
```
[อาชีพ, คุณสมบัติ, ความสำคัญ, เป้าหมาย, รายละเอียดแผน, -, -, -, -, -, -, -, -]
```

### 3. มีมากกว่า 3 แผน
- แสดงเฉพาะ 3 แผนแรก
- แผนที่เหลือจะไม่แสดง

### 4. แหล่งอ้างอิงหลายรายการ
- แสดงในเซลล์เดียวกัน
- แยกด้วย line break (`\n`)

### 5. ผลการประเมินหลายครั้ง
- แสดงในรูปแบบ: `วันที่: ค่า`
- แยกด้วย line break (`\n`)
- ตัวอย่าง:
  ```
  15/01/2026: 3
  20/01/2026: 4
  25/01/2026: 5
  ```

```javascript
treeData.forEach(careerNode => {
  careerNode.children.forEach(qualNode => {
    // ดึงข้อมูล
    const careerName = careerNode.label;
    const qualName = qualNode.label;

    qualNode.children.forEach(child => {

      if (child.header === 'target') target = child.label.replace('🎯 ค่าเป้าหมาย: ', '');
    });
    
    // จัดกลุ่มแผน
    const planGroups = {};
    // ... (ดูโค้ดเต็มในไฟล์)
    
    // สร้างแถว
    const row = [careerName, qualName, importance, target, ...];
    worksheet.addRow(row);
  });
});
```

## การแก้ไขปัญหา (Troubleshooting)

### ปัญหา: ข้อมูลไม่ครบ
**สาเหตุ:** โครงสร้างต้นไม้ยังไม่โหลดเสร็จ
**แก้ไข:** ตรวจสอบว่า `this.treeNodes` มีข้อมูลครบถ้วน

### ปัญหา: แผนไม่ตรงกับการประเมิน
**สาเหตุ:** การจับคู่ planTitle ไม่ตรงกัน
**แก้ไข:** ตรวจสอบชื่อแผนในโครงสร้างต้นไม้

### ปัญหา: ไฟล์ไม่ดาวน์โหลด
**สาเหตุ:** Browser block download
**แก้ไข:** ตรวจสอบ console และอนุญาต download

### ปัญหา: ข้อความแสดงผิด
**สาเหตุ:** Encoding issue
**แก้ไข:** ExcelJS รองรับ UTF-8 อัตโนมัติ ไม่ควรมีปัญหา
